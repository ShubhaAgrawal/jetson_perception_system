# Real-Time 3D Perception System — Jetson Orin Nano Super

Real-time 3D object detection, tracking, and occupancy grid generation on NVIDIA Jetson Orin Nano Super with OAK-D Lite.

**Status:** Work in progress

## Tech Stack
- NVIDIA Jetson Orin Nano Super (JetPack 6.1)
- ROS2 Humble
- YOLOv8n + TensorRT
- OAK-D Lite (RGB + Stereo Depth)
